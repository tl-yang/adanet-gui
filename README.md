### Install Dependencies
```pip install -r requirements.txt```  
### Run
```python3 main.py```

### Download Dataset
In this project, I used CIFAR-10, MNIST, Fashion MNIST to demonstrate. Dataset will be downloaded in the GUI, no addtional operation needed.
